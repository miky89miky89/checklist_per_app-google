# Checklist per app (Google Play)

Checklist riutilizzabile (IT/EN) per ridurre rifiuti in review Google Play, con focus su permessi/API “sensibili” (es. Accessibility API).

## Contenuti
- `PLAY_CONSOLE_PRE_SUBMIT_CHECKLIST.md` – checklist operativa prima di inviare una release

## Come usarla
1. Copia la checklist nella repo del progetto (o linkala nel tuo task/issue).
2. Compilala **prima** di caricare l’AAB e **prima** di inviare per revisione.
3. Se usi API sensibili, prepara in anticipo: disclosure, consenso, video demo, testo store listing.
